# Assis
Biggest Sync Library on Nodejs

## Includes
 - Encryption
 - Compression
 - Temprature Conversions
 - Date formatting

## Examples

###Setup Module
1. `$ npm i Assis@latest --save`
3. on code: `var assis = require('assis');`

###Encrypt
1. `assis.string.cryptography.encrypt('aes256', 'abc', 'foobar');`
